/r/Jailbreak Groupchat

1. NSFW (not safe for work) content is not allowed.

2. Trolling (such as spamming) will not be tolerated.

3. Keep conversation on controversial topics to a minimum.

4. Respect the opinions of others.

5. Hate talk (such as racial slurs or name calling) is forbidden.

6. Piracy is not allowed.

7. Be a kind person. Have fun and enjoy yourself.


If you have any questions or concerns, please contact an admin via a direct message.

If an admin does not notice a rule being broken, and you do, please report it via a direct message to an admin.

The first time a user breaks a rule, the user will be given a warning.

If the user breaks a rule again, the user will be removed for one day.

After breaking a rule for the third time, the user will be permanently banned.

You can disable Telegram notifications just for this chat by clicking on the top bar and choosing Mute.

Mentions (using @someusername) and replies (click and hold on a message, then select reply) will notify a user even if that person has notifications turned off for the chat.

You can tell who is an admin by looking at the group info.
